# Kanameliser Blender Extensions

## How to Add This Repository

1. Open Blender and go to **Edit > Preferences > Get Extensions**
2. Open the **Repositories** dropdown in the top right
3. Click the **+** button and select **Add Remote Repository**
4. Enter the following URL:

```
https://kxn4t.github.io/blender-extensions/index.json
```

5. Once the repository is added, you can search and install addons from the Get Extensions screen

> [!TIP]
> You can also install by dragging and dropping the download link below into the Blender window. The repository will be added automatically.
