# Kanameliser Blender Extensions

## リポジトリの追加方法

1. Blender を開き **Edit > Preferences > Get Extensions** に移動
2. 右上の **Repositories** ドロップダウンを開く
3. **+** ボタンから **Add Remote Repository** を選択
4. 以下の URL を入力:

```
https://kxn4t.github.io/blender-extensions/index.json
```

5. リポジトリが追加されると、Get Extensions 画面からアドオンを検索・インストールできます

> [!TIP]
> 下記アドオンのダウンロードリンクを Blender のウィンドウにドラッグ＆ドロップするだけでもインストールできます。リポジトリの追加も自動で行われます。
